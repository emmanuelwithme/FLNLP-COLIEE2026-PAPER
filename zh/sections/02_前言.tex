\section{前言}

法律判決檢索（legal case retrieval）是法律資訊系統中的核心任務之一。在 Competition on Legal Information Extraction and Entailment（COLIEE）Task~1 中，系統必須對每個查詢案例（query case）自動找出其對應的 noticed cases；若某案例曾被 query case 引用（referenced），則視為該 query 的 noticed case。為了真正測試模型能否從案件事實、推理與法律語意推斷支撐判決的先例，官方已將所有案例內文中的引用資訊移除（redacted）。資料來源為加拿大聯邦法院（Federal Court of Canada）判例，整個流程從輸入 query 到輸出 noticed cases 都必須完全自動化完成。

本研究的起點來自 THUIR@COLIEE 2023 的觀察 \cite{li2023thuir}。根據本研究表~\ref{tab:thuir-table3-excerpt} 節錄的數值 \cite{li2023thuir}，當 SAILER 與 BM25 都被限制為 512 tokens 時，SAILER 優於 BM25；但當 BM25 不受最大長度限制時，又反而高於受截斷的 SAILER。這表示法律檢索中的相對優劣不只取決於模型架構，也取決於模型究竟能看到多少判決全文。

COLIEE 2026 本身就是明顯的長文本任務。表~\ref{tab:length-stats} 顯示，7,708 篇清理後的整篇判決書文件平均長度為 5,031.36 tokens，中位數為 3,684.5 tokens，其中 56.11\% 短於 4096 tokens、87.69\% 短於 8192 tokens、94.80\% 短於 12,288 tokens。這也直接支持後續的 chunk-based reranking 設計：若將每篇文件最多切成 3 個 4096-token chunks，就能在不提高 encoder 上限的情況下，近乎完整覆蓋大多數文件。

表~\ref{tab:thuir-table3-excerpt} 進一步把這個問題說得更具體 \cite{li2023thuir}。當最大長度限制為 512 tokens 時，SAILER 的 F1 為 0.1385，高於 BM25 的 0.1012；但當 BM25 不受長度限制時，其 F1 上升到 0.1541，反而高於受截斷的 SAILER。本文的實驗則更進一步證明，只要把 encoder 的可見上下文拉長到 4096 tokens，以嵌入為基礎的檢索模型（embedding-based retrieval，也就是常說的 dense retrieval）就可以超過不限長度的 BM25：如表~\ref{tab:main-results-valid} 所示，經過 domain-adaptive pretraining（DAPT）與 supervised fine-tuning（SFT）的 ModernBERT 在 2026 valid 上達到 0.1650 F1，高於 BM25 的 0.1499。

基於上述觀察，本文採用原生支援 8192 tokens 的 ModernBERT \cite{warner2024modernbert} 作為長文本 encoder，並在美國判決書上進行 domain-adaptive pretraining（DAPT），也就是以 masked language modeling（MLM）進行 continued pretraining。然而，實務上的主要瓶頸很快轉為硬體限制：在單張 NVIDIA RTX 4080 SUPER 16GB 上，正式實驗只能使用 4096 tokens。此外，受限於硬體限制，DAPT 約訓練 60 小時，也只覆蓋 0.384 epoch，沒有完整看過全部 673 萬篇美國判決書。

本文有三個主要貢獻。第一，我們以 COLIEE 2026 重新檢驗長文本假設，並說明 query 表示方式與可見上下文長度的交互作用。第二，我們證明法律判決以嵌入為基礎的檢索模型能否穩定收斂，關鍵主要在於負樣本選取；靜態 BM25 hard negatives 幾乎無法訓練，動態負樣本則能穩定訓練，而年份範圍過濾提供較小但一致的額外收益。第三，我們把 embedding-based retrieval、scope filtering、LightGBM learning-to-rank（LTR），以及以 dynamic cut-off 為主要關鍵一環的 post-processing 整合成完整系統，最終在 COLIEE 2026 Task~1 官方結果中取得第 7 名 \cite{coliee2026task1results}。
